# simpler-windows11
Powershell script to simplify windows 11 VM to make it a good one again

🧠 这个脚本会做什么

✔ 关闭 SysMain
✔ 关闭 Windows Search
✔ 关闭 Delivery Optimization
✔ 关闭 Xbox 服务
✔ 关闭动画 + 透明效果
✔ 禁用后台应用
✔ 禁用 Widgets
✔ 删除 Web Experience Pack（解决 msedgewebview2.exe）
✔ 禁止 Search Web
✔ 降低 Microsoft Defender Antivirus 影响
✔ 添加 目标项目 排除目录 (在65行以后修改，添加)


🧠 What this script will do:

✔ Disable SysMain

✔ Disable Windows Search

✔ Disable Delivery Optimization

✔ Disable Xbox service

✔ Disable animations and transparency effects

✔ Disable background apps

✔ Disable Widgets

✔ Remove Web Experience Pack (fixes msedgewebview2.exe)

✔ Disable Web Search

✔ Reduce the impact of Microsoft Defender Antivirus

✔ Add target folder exclusion list (add/modify after line 65)
